# Inventeerimine

Inventeerimine is a GLPI 11 plugin designed to simplify IT asset inventory management.

It provides fast and structured inventory workflows, allowing users to search, update, and register assets while automatically tracking inventory dates.

Key features:
- Asset search by name, serial number, or inventory number
- Quick inventory update with automatic inventory date tracking
- Ability to add new assets when not found in system
- Inventory report with filtering by date range and asset type
- User and location assignment during inventory process
- Multilingual support (Estonian and English)

Supported asset types:
Computer, Monitor, NetworkEquipment, Peripheral, Phone, Printer, SoftwareLicense, PassiveEquipment, Enclosure, PDU

The plugin integrates directly into the GLPI Tools menu and follows native GLPI permissions and CSRF protection standards.

Designed for simplicity, speed, and accurate inventory tracking in enterprise IT environments.
